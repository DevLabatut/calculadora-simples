# Calculadora Simples

Projeto simples de calculadora desenvolvido com HTML, CSS e JavaScript. Permite realizar operações matemáticas básicas diretamente no navegador.

## Funcionalidades

* Soma de dois números
* Subtração de dois números
* Multiplicação de dois números
* Divisão de dois números
* Validação de campos vazios
* Limpeza automática dos campos após cada cálculo

## Tecnologias

* HTML
* CSS
* JavaScript

## Estrutura do Projeto

```
calculadora-simples/
│
├─ index.html
├─ css/
│  ├─ reset.css
│  └─ styles.css
└─ js/
   └─ script.js
```

## Como usar

Clone o projeto:

```
git clone https://github.com/DevLabatut/calculadora-simples.git
```

Abra o arquivo `index.html` no navegador.

Digite dois números na operação desejada e clique no botão correspondente para ver o resultado.
